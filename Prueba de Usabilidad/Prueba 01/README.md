## 2. RESULTADOS DE LA PRUEBA DE USABILIDAD

| ID ITERATION | ID USABILITY TESTING | FECHA      |
| :----------- | :------------------- | :--------- |
| 01           | 01                   | 03/11/2025 |

---

En la prueba de usabilidad se obtuvo lo siguiente:

- En el prototipo se usó el primer enfoque que es el **pomodoro**. Se abrió la interfaz de manera eficaz.
- Se probó la interacción con la interfaz.

### Retro (Retroalimentación)

- Agregar un **Botón de pausa**.
- El usuario acordó que el símbolo de tuerca debería ser un poco más específico, ya que se le preguntó que, si gusta cambiar el rango de tiempo de **pomodoro**, ¿Cómo lo haría? Por lo que el usuario, no pudo hacerlo ya que el icono de configuración de **tiempo**, no es intuitivo.
- Al concluir, el usuario comentó que la interfaz está agradable, salvo el detalle anterior.

### Hallazgos y Lista de Revisión (Review List)

Se identificaron los siguientes puntos de acción basados en la retroalimentación del usuario:

| ID  | Observación                                                                                   | Clasificación |
| :-- | :-------------------------------------------------------------------------------------------- | :------------ |
| 01  | Agregar un Botón de pausa.                                                                    | FR            |
| 02  | El icono de configuración de tiempo (tuerca) no es intuitivo para cambiar el rango de tiempo. | B             |
