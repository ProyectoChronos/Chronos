# Reporte de Prueba de Usabilidad: Prototipo Pomodoro

Este documento resume los hallazgos de la sesión de pruebas de usabilidad (ID Iteración: 01) realizada el 03/11/2025.

## Resumen de la Sesión

Se probó la interacción con el primer prototipo enfocado en el método Pomodoro.

### Puntos Positivos

* La interfaz se abrió de manera eficaz.
* Se probó la interacción inicial con la interfaz.
* Al concluir, el usuario comentó que **la interfaz es agradable** visualmente.

---

## Hallazgos y Lista de Revisión (Review List)

Se identificaron los siguientes puntos de acción basados en la retroalimentación del usuario:

| ID | Observation | Classification* |
| :--- | :--- | :--- |
| 01 | Agregar un Botón de pausa. | FR |
| 02 | El icono de configuración de tiempo (tuerca) no es intuitivo para cambiar el rango de tiempo. | B |

---

### *Clasificación

* **B = Bug**: Error a ser corregido en la iteración actual.
* **FR = Feature Required**: Funcionalidad requerida a ser considerada en el *project backlog*.
